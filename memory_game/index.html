Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-sijaku, Pexeso or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to tun over pairs of matching cards.

Why is it fun?

Not only is this game a blast, but it tests your memory and helps you improve it!
